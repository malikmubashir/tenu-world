## Zone tendue logic
- Postal code extracted from address via regex
- Client-side: département-level check for instant UI feedback
- Server-side: full 1,100+ postal code lookup from décret annex
- Notice period: meublé always 1 month, non-meublé zone tendue 1 month, non-meublé elsewhere 3 months
- Deposit return: 1 month (matching) or 2 months (deductions) per Article 22
